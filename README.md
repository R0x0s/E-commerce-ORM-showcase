Eccomerce Backend 



Features
Easy-to-use back-end application.
Manages database of online store.

##Prequisites
Install Node.js
Install MySQL2

##Usage
On the command line do "npm start"
Open Insomnia Core, Postman, or a similar application.
Make API request

##Technologies Used
JavaScript
Node.js
Express.js
MySQL2
Sequelize
Dotenv
